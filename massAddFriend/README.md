# massAddFriend
Send mass friend requests to server members.

## Installation
NpmJS:
```
npm i discord.js-selfbot-v13 bottleneck
```

## Usage
```
node index.js <token> <guildID>
```

- token - Your account Discord token.
- guildID - The target server ID.
